# PlayJazz CRM

## Overview
A React-based CRM dashboard application for music education management. Built with React 19, TypeScript, and Vite.

## Project Structure
- `components/` - React components (Dashboard, Financial, SalesCRM, Settings, Students)
- `services/` - API services
- `App.tsx` - Main application component
- `types.ts` - TypeScript type definitions
- `constants.ts` - Application constants

## Tech Stack
- React 19
- TypeScript
- Vite
- Recharts (charts)
- Lucide React (icons)
- Tailwind CSS (via CDN)

## Development
- Dev server runs on port 5000
- Run: `npm run dev`
- Build: `npm run build`

## Deployment
- Static deployment
- Build output: `dist/`
